使用Maven构建

构建时跳过单元测试

在pom.xml加入以下插件, skip = true

构建时忽略单元测试错误

在pom.xml加入以下插件, testFailureIgnore = true

使用maven对wildfly进行远程部署

本地配置

在pom.xml加入以下插件, 使用 mvn wildfly:deploy 命令进行部署, mvn wildfly:undeploy 命令移除部署

远程配置

1、监听端口，默认为 9990

2、创建management用户, 使用自带的 add-user.sh 或者 add-user.bat
